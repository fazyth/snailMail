# UnlessAPP

## Hackathon Project Prompt

Make the most hilariously useless app possible.

### Useless Email Service

An email service that sends an email to the recipient but it takes the same time
to deliver as a traditional postal service. It calculates the distance between
sender and recipient and calculates the delivery time based on walking that distance.
